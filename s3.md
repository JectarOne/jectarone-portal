# JectarOne Portal
## Sprint 3 — Findings & Evidence System

**Status:** Ready to Build  
**Priority:** Critical  
**Estimated Progress After Completion:** ~45% of MVP

---

# Objective

Transform the portal from an assessment tracker into a real cybersecurity assessment platform.

After Sprint 3, consultants should be able to document vulnerabilities, attach evidence, calculate risk, and prepare professional reports.

This sprint becomes the foundation for PDF generation, dashboards, AI assistance, remediation tracking, and client collaboration.

---

# User Story

As a security consultant,

I want to create findings inside an assessment,

so that I can produce a complete professional security report for my client.
